# Rapports de nettoyage

Rapports dry-run et apply pour suppressions validées (promotions inutiles, scams avérés, réseaux sociaux > 30 jours, doublons).

Toute nouvelle règle destructive commence par un dry-run listant les mails concernés.

Voir [docs/07_RETENTION_AND_DELETION_RULES.md](../../docs/07_RETENTION_AND_DELETION_RULES.md).
